5. Projects

Warning

Przedstawione niżej screeny układu plików pochodzą z exploratora windows, u ciebie może to wyglądać inaczej

5.1 Main

Główna podstrona jest generowana przez plik projects.php znajdujący się w głównym katalogu:

Header, company i footer są generowane tak jak opisano w Index. Main dla strony głównej składa się z 1 sekcji:

  • Lista projektów

Kody w pliku projects.php odpowiadające każdej z części:

<!-- Galleries -->
<?php include('includes/projects/research_projects.php'); ?>

5.1.1 Kafelek z projektem

Każdy z kafelków z opisem projektu generowany jest przez plik includes/projects/research_projects.php, który wygląda następująco

<div class="row center-xs center-sm center-md center-lg  middle-xs middle-sm middle-md middle-lg">
    <div class="col-xs-12 col-sm-12 col-md-12 col-lg-12">
        <h1>Research <span class="primary-text">Projects</span></h1>  
    </div>
</div>
<hr>
<div class="row center-xs center-sm center-md center-lg middle-xs middle-sm middle-md middle-lg ">
    <!-- Project start -->
    <?php
        $title = "Slow, non-diffusive dynamics in quantum many–body systems";
        $link = "https://www.archiwum.nauka.gov.pl/projekty-i-inicjatywy/diamentowy-grant.html"; // set to true if Head of Department
        $type = "NCN OPUS 2020/37/B/ST3/00020"; //set to true if Vice-head of Department
        $dates = "2020-2025"; // if not exist set false
        $NameSurname = "Marcin Mierzejewski";
        include('Project.php');
    ?>
    <?php
        $title = "Krytyczne fazy materii bez bariery energetycznej, od topologii do geometrii";
        $link = false; // set to true if Head of Department
        $type = "SONATA BIS"; //set to true if Vice-head of Department
        $dates = "2020-2025"; // if not exist set false
        $NameSurname = "Piotr Surówka";
        include('Project.php');
    ?>
    <!-- Project end -->
    ...
    <!-- Project start -->
    <?php
        $title = "Zderzenia ścian domenowych i kreacja periodycznych struktur domen dla zapisu i przetwarzania informacji";
        $link = "https://ncn.gov.pl/sites/default/files/pliki/uchwaly-rady/2015/uchwala43_2015-zal3.pdf";  
        $type = "NCN HABILITACYJNY"; 
        $dates = "2010-2013"; 
        $NameSurname = "Andrzej Janutka";
        include('Project.php');
    ?>
    <!-- Project end -->
</div>
Każdy kafelek to wewnętrzny kod php, który zawiera zestaw zmiennych

  • $title - określa tytuł projektu (preferowany język angielski).
  • $link - określa link do stron zewnętrznych (na przykład NCN) z informacjami o projekcie.
  • $type - określa rodzaj projektu i numer projektu.
  • $dates - określa lata w jakich realizowany jest projekt.
  • $NameSurname - określa imię i nazwisko kierownika projektu.

Taki zestaw zmiennych ładowany jest przez plik includes/projects/Project.php i generuje odpowiedni kafelek

5.2 Jak wprowadzać zmiany

5.2.1 Dodawanie projektu

Aby dodać projekt należy w pliku includes/projects/research_projects.php skopiować kod 

<?php
    $title = "Zderzenia ścian domenowych i kreacja periodycznych struktur domen dla zapisu i przetwarzania informacji";
    $link = "https://ncn.gov.pl/sites/default/files/pliki/uchwaly-rady/2015/uchwala43_2015-zal3.pdf";  
    $type = "NCN HABILITACYJNY"; 
    $dates = "2010-2013"; 
    $NameSurname = "Andrzej Janutka";
    include('Project.php');
?>
zmodyfikować zmienne $title, $link, $type, $dates oraz $NameSurname (znaczenie zmiennych tutaj: link) i dokleić na końcu pliku przed zamykającym </div>

5.2.2 Usuwanie galerii

Aby usunąć grupę wystarczy usunąć wszystkie pliki i kody wspomniane wyżej w Dodawanie projektu
